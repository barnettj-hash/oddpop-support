# OddPop Support

Static support website for OddPop Arcade.

## GitHub Pages

Recommended repository:

```text
barnettj-hash/oddpop-support
```

Recommended App Store URLs:

```text
https://barnettj-hash.github.io/oddpop-support/
https://barnettj-hash.github.io/oddpop-support/support.html
https://barnettj-hash.github.io/oddpop-support/privacy.html
```

## Files

- `index.html` is the public support homepage.
- `support.html` is the dedicated support-request page for App Store Connect.
- `privacy.html` is the privacy policy.
- `support.css` contains the neon support-site styling.
- `oddpop-app-icon.svg` is the local site icon.

## Publish

In GitHub, enable Pages for the repository and publish from the main branch root.
